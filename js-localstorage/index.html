JavaScript localStorage에 객체 저장하기 (feat. 우테코 프리코스 3주차)

posted by Fry · December 12, 2021

우아한 테크코스 3주차 프리코스 미션을 구현하는데 등장한 낯선 localStorage..
제대로된 코드를 작성하기 전에 미리 localStorage를 공부하면서 필요한 객체를 저장하고 꺼내보는 연습을 하자!

👀 들어가기에 앞서..

우아한 테크코스 3주차 프리코스 미션은 자판기를 구현하는 것이다. 자판기는 크게 3개의 기능이 있는데,

  1. 상품 추가가 가능한 상품 관리 기능
  2. 자판기 동전 충전 기능
  3. 자판기로 상품을 구매할 수 있는 기능

이다.
자세한 내용은 미션 저장소를 통해 확인할 수 있다.


✅ 저장할 객체 생성

먼저 미션 구현을 위해 어떤 객체를 localStorage에 저장해야 하는지를 생각해봐야 한다.

  • 상품 관리 (Stock)

    • 상품명 (productName)
    • 가격 (price)
    • 수량 (remaining)

  • 자판기 잔돈 관리 (Change)

    • 500원 (coin500)
    • 100원 (coin100)
    • 50원 (coin50)
    • 10원 (coin10)

일단은 이정도 엔티티이면 충분할 것 같다! 자바스크립트 코드로 간단한 클래스를 작성해보자.

class Stock {
  constructor() {
    this.items = [];
  }

  addItem({ name, price, remaining }) {
    this.items.push({ name, price, remaining });
  }
}

const stock = new Stock();
stock.addItem({ name: '콜라', price: 1200, remaining: 20 });
class Change {
  constructor() {
    this.coins = {
      500: 0,
      100: 0,
      50: 0,
      10: 0,
    };
  }

  push(size, count) {
    this.coins[size] += count;
  }
}

const change = new Change();
change.push(500, 3);
change.push(50, 2);

❓ localStorage 이해하기

localStorage는 웹 스토리지 객체(web storage object) 중 하나이다. localStorage를 사용해서 키-값 쌍의 데이터를 저장해두면, 페이지를 새로 고침하거나 브라우저를 재실행해도 데이터가 사라지지 않고 남아있다.

참고로 Storage에 저장된 데이터는 관리자 도구 > Application > Local Strorage 에서 확인할 수 있다.

이제 위에서 작성한 객체들을 localStorage에 저장해보자!

1. 데이터 저장하기

localStorage에 데이터를 저장하는 방법은 일반 객체와 비슷하다. 크게 3가지 방법이 있다.

localStorage.key = value;
localStorage[key] = value;
localStorage.setItem(key, value);

객체 데이터 저장하기 - JSON.stringify()

localStorage에 데이터를 저장할 때 주의할 점이 있는데, 바로 키와 값은 모두 문자열이어야 한다는 것이다.

localStorage.setItem('Stock', stock); // (X) [object Object]
localStorage.setItem('Change', change); // (X) [object Object]

객체 데이터를 그대로 저장하게 되면, 아래 사진처럼 값이 제대로 저장되지 않는다.

따라서 객체를 저장할 때는 반드시 JSON.stringify() 로 객체를 문자열로 바꾼 후 저장해야한다.

localStorage.setItem('Stock', JSON.stringify(stock));
localStorage.setItem('Change', JSON.stringify(change));

2. 데이터 불러오기

저장할 때 설정한 key로 값을 얻을 수 있다. 이 때 getItem 메서드를 사용한다.

localStorage.getItem(key);

객체 데이터 불러오기 - JSON.parse()

앞에서 언급했듯이 스토리지에 저장된 값은 문자열이다. getItem으로 읽어온 데이터 또한 문자열이다. 이를 자바스크립트 객체로 사용하기 위해서는 JSON.parse() 를 통해 문자열을 object로 전환하는 과정이 필요하다.

const stockItems = JSON.parse(localStorage.getItem('Stock'));
const coins = JSON.parse(localStorage.getItem('Change'));

3. 데이터 삭제하기

localStorage의 데이터 삭제는 removeItem 메서드를 통해 가능하다.

localStorage.removeItem(key);
localStorage.removeItem('Stock');
localStorage.removeItem('Change');

💬 마무리하며

localStorage를 다루는 것은 생각보다 간단했다. (처음 접해 본 개념이라 너무 겁먹었던 것 같다.. ^_^)
하지만 localStorage를 공부하다보니 문득 이런 생각이 들었다.

데이터 조그마한 부분을 수정하는데도 데이터 객체를 전부 통째로 다시 덮어써야하는건가? 🧐

일단 localStorage에 저장된 값을 수정하고 싶다면, 값을 바꿔서 덮어 씌우는 방법 밖에 없다.
이러한 비효율성을 줄이고자 한다면, 객체를 작은 부분으로 나눠 저장하는 것도 하나의 방법이 될 것 같다.
이 부분은 미션을 구현하면서 고민해봐야겠다!
